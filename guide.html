🔐 账号安全

重要提醒

永远不要向任何人提供您的 ChatGPT 密码！

安全建议

1. 使用强密码

  • • 至少12位
  • • 包含大小写字母、数字、特殊字符
  • • 定期更换

2. 开启两步验证

  • • 设置 → 安全 → 两步验证
  • • 使用 Google Authenticator

3. 定期检查登录记录

  • • 查看异常登录
  • • 及时修改密码

4. 避免共享账号

  • • 不要多人使用同一账号
  • • 避免在公共电脑登录

🔧 故障排除

Q1: 充值后没有看到 Plus 标识

解决方案：

  1. 1. 退出账号重新登录
  2. 2. 清除浏览器缓存
  3. 3. 换个浏览器试试
  4. 4. 联系客服核实

Q2: 无法使用 GPT-4

解决方案：

  1. 1. 确认 Plus 标识存在
  2. 2. 在对话框上方切换模型
  3. 3. 检查是否达到使用限制

Q3: 插件无法使用

解决方案：

  1. 1. 设置 → Beta features → 开启插件
  2. 2. 新建对话后启用插件
  3. 3. 最多同时启用3个插件
